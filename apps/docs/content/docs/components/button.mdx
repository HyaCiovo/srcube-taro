# Button

Buttons allow users to trigger an action or event with a single tap.

## Installation

<PackageInstall commands={{ 
  npm: 'npm install @srcube-taro/button',
  yarn: 'yarn add @srcube-taro/button',
  pnpm: 'pnpm add @srcube-taro/button',
}} />

## Import

```tsx
import { Button } from '@srcube-taro/ui'
```
## Usage

### Colors

Button comes with 5 different colors: `default`, `primary`, `success`, `warning`, and `danger`.

```tsx
<Button>Default Button</Button>
<Button color="primary">Primary Button</Button>
<Button color="success">Success Button</Button>
<Button color="warning">Warning Button</Button>
<Button color="danger">Danger Button</Button>
```
### Variants

Button supports 5 style variants: `solid`, `outline`, `flat`, `text`, and `shadow`.

```tsx
<Button variant="solid">Solid Button</Button>
<Button variant="outline">Outline Button</Button>
<Button variant="flat">Flat Button</Button>
<Button variant="text">Text Button</Button>
<Button variant="shadow">Shadow Button</Button>
```
### Sizes

Button comes in 4 different sizes: `xs`, `sm`, `md`, and `lg`.

```tsx
<Button size="xs">Extra Small</Button>
<Button size="sm">Small</Button>
<Button size="md">Medium</Button>
<Button size="lg">Large</Button>
```

### Loading State

Use the `loading` prop to show a loading spinner.

```tsx
<Button loading>Loading</Button>
<Button loading spinnerPlacement="end">Loading</Button>
```

### Disabled State

Use the `disabled` prop to disable button interactions.

```tsx
<Button disabled>Disabled Button</Button>

### Block Button

Use the `block` prop to make the button fill the full width of its container.
```

```tsx
<Button block>Block Button</Button>

### With Icons

Use `startContent` and `endContent` to add icons or other content before or after the button text.

tsx
<Button startContent={<Icon name="plus" />}>Add Item</Button>
<Button endContent={<Icon name="arrow-right" />}>Next</Button>
```

### Styling

The Button component uses Tailwind CSS for styling and supports the following class variants:

- Base styles include flexbox layout, font weight, transitions, and border styles
- Color variants apply appropriate background colors and text colors
- Size variants control height, padding, font size, and border radius
- Variant styles modify the appearance with different background, border, and shadow combinations
- Hover and active states provide visual feedback for user interactions
