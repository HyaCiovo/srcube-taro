# Introduction

## What is SrcubeUI Taro?

SrcubeUI Taro is a UI library for Taro.js that helps you build beautiful and modular cross-platform mini app user interfaces.
Create on top of [Tailwind CSS](https://tailwindcss.com/) with [Weapp Tailwind CSS](https://tw.icebreaker.top/) and [Taro Components](https://taro-docs.jd.com/docs/components-desc).
